HTML

HTML (Hypertext Markup Language) é a linguagem de marcação padrão usada para criar páginas web, estruturando seu conteúdo com elementos como títulos, parágrafos e links.

CSS

CSS (Cascading Style Sheets) é uma linguagem usada para definir a aparência visual e o layout de páginas web, controlando elementos como cores, fontes e posicionamento. Ele separa a estrutura HTML da estilização, permitindo um design flexível e responsivo.

Javascript

JavaScript é uma linguagem de programação usada para criar interatividade em páginas web, permitindo manipulação dinâmica de conteúdo, validação de formulários e animações. Ele executa funções no lado do cliente (navegador) e pode se comunicar com servidores.

React

React é uma biblioteca JavaScript para construir interfaces de usuário, permitindo a criação de componentes reutilizáveis que atualizam de forma eficiente quando os dados mudam. Ele utiliza uma abordagem declarativa e é mantido pelo Facebook

Node

Node.js é um ambiente de execução JavaScript no lado do servidor, permitindo que desenvolvedores usem JavaScript para criar aplicativos de backend. Ele é baseado no motor V8 do Google Chrome e é eficiente para manipulação de I/O e construção de servidores escaláveis.
